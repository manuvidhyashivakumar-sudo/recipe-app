# Recipe App

A modern Recipe Application built using React JS and TailwindCSS.

## Features

- Browse recipes
- Search recipes
- Filter by category
- View recipe details
- Add/Remove favorites
- Persistent favorites using localStorage
- Responsive UI

## Tech Stack

- React JS
- TailwindCSS
- Axios
- React Router DOM
- MealDB API

## API Used

https://www.themealdb.com/api.php

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed using Netlify.
